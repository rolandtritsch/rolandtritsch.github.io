--- layout: post title: Effective Android - Using Apache CXF to access SOAP services --- A year ago I developed a small set of demos to show how to access SOAP services from the iPhone. To do this I used a very interesting feature of Apache CXF: The ability to generate a complete JavaScript client-side SOAP-stack on the fly.

I just finished an update/upgrade of this demo to show that it also works with the Android Phone. I also ported it to maven, means installing, building and running the demo is much easier now. The source code and the maven build environment is available on GitHub.

Enjoy :).
